# MLP Fingerspelling Model

This directory references the static fingerspelling recognition model developed as an independent component of the SignVox project.

The model is trained on the **IEEE DataPort ASL Fingerspelling Dataset** using a lightweight Multilayer Perceptron (MLP).

To avoid duplication, the complete implementation, training pipeline, and pretrained weights are maintained in a dedicated repository.

## Repository

[SIGN2SOUND_ZORA](https://github.com/basil-saji/SIGN2SOUND_ZORA)

The standalone repository contains:

* Training notebooks
* Dataset preprocessing
* Model checkpoints
* Evaluation results
* Documentation
